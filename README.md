# scapy-iscsi

iSCSI layer for Scapy.

## Hacking

To start hacking do the following:

```
$ make venv
$ . venv/bin/activate
```

Take a look at examples/ to get the idea on how to use this library.

## Testing

Run the following commands to ensure that your changes are correct:

```
$ . venv/bin/activate
$ make checks
```
